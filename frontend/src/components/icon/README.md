## description

- icon을 보여주는 컴포넌트
- div에 svg mask를 적용하는 방식 사용

## props

|   name   | description | default |
| :------: | :---------: | :-----: |
|  `name`  |  icon name  |         |
| `color`  |    color    |         |
| `width`  |    width    |         |
| `height` |    width    |         |

## use case

```tsx
<Icon name={'home'} color="primary" width="3rem" height="3rem" />
```
