# Button

## description

- 기본 버튼 컴포넌트

## props

|   name    |       description       |  default  |
| :-------: | :---------------------: | :-------: |
|  `color`  |       text color        | 'gray50'  |
| `bgColor` | button background color | 'primary' |

## use case

```tsx
<Button>I'm here<Button>

<Button color='black' bgColor='warning'>I'm here<Button>

<Button disabled>I'm here<Button>
```
